# Escrow Quick Start - 5 Minutes

## For Tourists (Lock Funds)

### 1. Get Setup (2 minutes)
```
✅ Install Phantom: https://phantom.app/
✅ Get Devnet SOL: https://faucet.solana.com/
✅ Connect wallet to TourChain
```

### 2. Create Booking (1 minute)
```
1. Go to: http://localhost:3000/explore
2. Select a route
3. Click "Book Now"
4. Fill details and confirm
```

### 3. Lock Escrow (2 minutes)
```
1. On confirmation page, see "Escrow Panel"
2. Click "Lock X SOL in Escrow"
3. Approve in wallet popup
4. Done! ✅ Funds locked on-chain
```

---

## For Guides (Enable Escrow)

### 1. Link Wallet (2 minutes)
```
1. Go to: http://localhost:3000/dashboard/operator
2. Click "Select Wallet"
3. Choose Phantom/Solflare
4. Click "Link Wallet"
5. Sign message
6. Done! ✅ Escrow enabled
```

### 2. Receive Payments
```
Automatic milestone releases:
├─ 20% on booking confirmation
├─ 30% when trek starts
├─ 30% at halfway point
└─ 20% on completion
```

---

## Quick Troubleshooting

| Issue | Solution |
|-------|----------|
| "Connect wallet first" | Install Phantom, click "Select Wallet" |
| "Guide hasn't linked wallet" | Guide must link wallet in dashboard |
| "Insufficient funds" | Get devnet SOL from faucet |
| "Transaction failed" | Check wallet has enough SOL, try again |
