# Tourism Chain Nepal

Solana-Powered Decentralized Tourism Ecosystem for the Colosseum Hackathon.

## Structure
- \programs/\: Anchor programs (Tourism Registry, Booking Escrow, Experience NFT, Loyalty Token, DAO Governance, Route Registry)
- \pps/web/\: Next.js 14 App Router frontend
- \sdk/\: TypeScript SDK for program interactions
- \	ests/\: Integration tests

## Stack
- Solana + Anchor (Rust)
- Next.js 15 + Tailwind CSS 4
- Metaplex Bubblegum (cNFTs)
- Framer Motion
- Lucide React
