# How to Create TourChain PowerPoint Presentation

## Step-by-Step Guide

### Step 1: Download Template

**Option A: Use Microsoft PowerPoint**
1. Open PowerPoint
2. Search for "Startup Pitch Deck" template
3. Choose a modern template with purple/orange colors

**Option B: Use Google Slides**
1. Go to slides.google.com
2. Click "Template Gallery"
3. Choose "Pitch" template
4. Customize colors to match TourChain

**Option C: Use Canva (Easiest)**
1. Go to canva.com
2. Search "Pitch Deck"
3. Choose "Startup Pitch Deck" template
4. Customize with TourChain content

---

### Step 2: Set Up Brand Colors

**TourChain Color Palette**:
```
Primary Purple: #6366f1 (RGB: 99, 102, 241)
Secondary Orange: #f59e0b (RGB: 245, 158, 11)
Accent Green: #10b981 (RGB: 16, 185, 129)
Dark Navy: #1a2b4a (RGB: 26, 43, 74)
Light Background: #f8faff (RGB: 248, 250, 255)
```

**How to apply**:
1. Go to Design → Colors → Customize Colors
2. Set Accent 1: Purple (#6366f1)
3. Set Accent 2: Orange (#f59e0b)
4. Set Accent 3: Green (#10b981)
5. Save as "TourChain Theme"

---

### Step 3: Add TourChain Logo

**Get Logo**:
1. Go to http://localhost:3000
2. Right-click on logo
3. Save image as `tourchain-logo.png`

**Add to Slides**:
1. Insert → Picture → This Device
2. Select logo file
3. Resize to 150x50px
4. Place in top-left corner
5. Copy to all slides (Ctrl+C, Ctrl+V)

## Step 5: Add Animations (Optional)

**Recommended Animations**:
1. **Slide Transitions**: Fade or Morph
2. **Title Entrance**: Fly In from Left
3. **Content**: Fade In (0.5s delay)
4. **Charts**: Wipe or Grow
5. **Numbers**: Count Up effect

**How to Add**:
1. Select object
2. Animations tab → Add Animation
3. Choose effect
4. Set duration (0.5-1s)
5. Preview

---

## Step 6: Export

**For Presentation**:
- File → Save As → PowerPoint Presentation (.pptx)

**For Sharing**:
- File → Export → PDF
- File → Export → Video (for social media)

**For Web**:
- Upload to Google Slides
- Share link with view access
