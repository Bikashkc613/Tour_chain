**What to capture**:

## Screenshot Tips

### Quality:
- Use 1920x1080 resolution
- Full screen (F11) for clean shots
- Hide browser bookmarks bar
- Clear notifications

### Tools:
- **Windows**: Snipping Tool or Windows + Shift + S
- **Mac**: Command + Shift + 4
- **Chrome**: DevTools screenshot (Ctrl + Shift + P → "screenshot")

### Editing:
- Use Paint or Photoshop to crop
- Add arrows/highlights if needed
- Compress images (tinypng.com)
- Save as PNG for quality

---

## After Taking Screenshots

1. Create a folder: `TourChain-Screenshots`
2. Name files clearly: `01-homepage.png`, `02-explore.png`, etc.
3. Upload to Google Drive or Dropbox
4. Use in PowerPoint/Canva/Figma

## Alternative: Screen Recording

If screenshots aren't enough, record a video:

**Windows**:
1. Press Windows + G (Game Bar)
2. Click record button
3. Navigate through app
4. Save video

**Use for**:
- Product demo slide
- Showing escrow flow
- Interactive features
- User journey
